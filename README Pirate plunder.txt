README

Thank you for playing Pirate Plunder!

== GAME DESCRIPTION ==

This is a dungeon-crawler-like game where you're stranded on an island and have to survive multiple waves of enemies guarding their treasure! Defeat
them all and see how long you can survive!


== REQUIRED DOWNLOADS ==

*This game runs off of Python so that is required. To donwload the latest version of Python go to: https://www.python.org/downloads
	-At the time of release, the game runs on Python Version 3.8.5. If the game fails to launch for whatever reason, try running
	 on this version of Python.

*This game requires Arcade library in python in order to play. To download Arcade go to: https://arcade.academy/installation.html
	-At the time of release, the game runs on arcade version 2.4.3. If the game fails to launch for whatever reason, try running
	 on this version of Arcade.



== WHAT'S NEW IN THIS VERSION ==

The game is new!


== COMPATIBILITY ==

Pirate Plunder currently runs on Windows platforms. It has been tested on Windows 10.

                                            
== RUNNING THE GAME ==

To play Pirate Plunder, run the window.py file from the game directory. It is also possible to create a shortcut of this game and run it
from there.


== CONTROLS ==

KEYBOARD
WASD  Arrow keys	Movement
Mouse			Aim
Click			Shoot
